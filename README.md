## Installation
You need yarn installed

Node.js < 16.10, install corepack
```
npm i -g corepack
```

Node.js >= 16.10, enable corepack
```
corepack enable
```

Install packages
```
yarn install
```

## Usage
Go to packages/node-server and run
```
yarn start
```

Open another terminal instance, navigate to packages/react-app and run
```
yarn start
```
